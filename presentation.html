The premise

A guitar is two things: a string, and a box around the string.

Karplus-Strong gives us the string. Where's the box?

01 · Concept from class

Karplus-Strong is a circle.

A burst of filtered noise enters the delay line. Each sample is read out, low-pass filtered, scaled by a damping coefficient, and written back to the head of the buffer.

Pitch comes for free from the buffer length:

f₀ = sampleRate ÷ N

Decay is just loop gain. Brightness is the filter's α.

02 · Technique from class — sharpened

The native DelayNode lies above 340 Hz.

DelayNode minimum delay

128 samples ÷ 44,100 Hz ≈ 2.9 ms

Highest reachable f₀

1 ÷ 2.9 ms ≈ 344 Hz (≈ F4)

Above F4, every note drifts sharp.

worklet.js · process()
// runs sample-by-sample on the audio thread
const current = delayLine[readIndex];
lowpass = α * current + (1 - α) * lowpass;
const feedback = lowpass * damping;

channel[i] = current;
delayLine[readIndex] = feedback;
readIndex = (readIndex + 1) % bufferSize;

→  An AudioWorklet moves the loop into the audio thread, at sample resolution. No more out-of-tune treble.

03 · Technique not from class

We grow our own impulse response.

A real guitar body is a coupled resonator with a small set of strong vibrational modes. Each mode is, mathematically, just a decaying sinusoid — so the impulse response is the sum:

ir(t)  =  Σᵢ  aᵢ · e−t / τᵢ · sin(2π fᵢ t)

Hz τ Role
100 0.55 s Helmholtz air mode (soundhole)
196 0.32 s T(1,1)₂ — top plate, coupled
285 0.24 s T(1,1)₃ — back plate, coupled
425 0.16 s upper top-plate mode
610 0.12 s upper coupled mode
880 0.09 s body brilliance

No recorded samples shipped. The convolver eats this synthetic IR and the string voice gets a guitar's body for free.

What I'm most proud of.

  1. 01

    Polyphony with self-terminating voices

    Each AudioWorkletNode messages the main thread once it has decayed below −66 dB. The node is then disconnected and garbage-collected. Hold ten notes; the graph stays clean.

  2. 02

    A/B between body presets

    The same string voice runs through four different procedurally-modeled bodies — guitar, koto-like, banjo-like, raw. The body's contribution becomes audible by comparison.

  3. 03

    Mode data grounded in real acoustics

    The guitar preset uses the documented Helmholtz, T(1,1)₂, and T(1,1)₃ frequencies of a classical guitar body — not numbers chosen by ear.
